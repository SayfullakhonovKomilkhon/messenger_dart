# Product Context

## Purpose
Мессенджер для личного общения через текстовые сообщения, обмен файлами и аудио/видеозвонки.

## User Experience Goals
- Мгновенная доставка сообщений через WebSocket
- Надежная аутентификация с автоматическим обновлением токенов
- Поддержка медиа (фото, видео, аудио, PDF)
- Качественные звонки через WebRTC + TURN сервер

## How It Works
1. Пользователь регистрируется по номеру телефона
2. Ищет собеседников по имени или username
3. Создает диалог и обменивается сообщениями в реальном времени
4. Может совершать аудио/видео звонки
5. Может прикреплять файлы к сообщениям

## API Access
- HTTP API: /api/v1/* с JWT Bearer auth
- WebSocket STOMP: /ws с JWT в STOMP headers
- Flutter клиент: http://10.0.2.2:3000 (Android), localhost:3000 (iOS)
